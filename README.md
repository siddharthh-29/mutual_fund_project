## 📌 Project Overview
A comprehensive end-to-end data engineering and quantitative analysis project focused on the ₹68 Trillion Indian Mutual Fund industry. This project transforms raw, fragmented investor and fund data into an interactive Power BI executive ecosystem supported by Python-driven institutional risk metrics.

## 🏗️ Architecture & Tech Stack
* **Data Engineering:** Python (`pandas`, `numpy`) for ETL, temporal standardization, and handling ragged data hierarchies.
* **Quantitative Analytics:** Python (`scipy`, `matplotlib`) for Historical Value at Risk (VaR), Conditional VaR (CVaR), Rolling 90-day Sharpe Ratios, and Herfindahl-Hirschman Index (HHI) concentration metrics.
* **Business Intelligence:** Power BI (Relational Star Schema, DAX, Dual-Axis Tracking, Dynamic Matrix Heatmaps).
